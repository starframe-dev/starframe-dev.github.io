# Лендинг-визитка Starframe

## Контекст

Starframe — организация, которая создаёт кодовые библиотеки, инструменты и проекты. Нужен простой статический сайт-визитка, который рассказывает, что организация существует и чем занимается. Конкретные проекты показывать не нужно, т.к. всё в разработке.

## Цель

Создать одностраничный статический лендинг на Astro в папке `/Users/a/Space/Projects/Starframe/website` с базовой информацией об организации и ссылкой на GitHub.

## Что изменится

1. `/Users/a/Space/Projects/Starframe/website/` — папка сайта
2. `astro.config.mjs` — конфигурация Astro со статическим выводом (`output: 'static'`)
3. `package.json` — зависимости Astro + TypeScript
4. `src/pages/index.astro` — единственная страница лендинга
5. `src/components/sections/` — Hero, About, Features, GitHub, Footer
6. `src/styles/global.css` — стили: градиентный фон, анимации, адаптив
7. `public/` — favicon

## Детали реализации

1. Инициализировать проект Astro (базовый шаблон).
2. Включить статический вывод в `astro.config.mjs`.
3. Оставить только одну страницу `index.astro`.
4. Структура страницы:
   - Hero: название `Starframe` с glow-эффектом, слоган «Создаём проекты, библиотеки и инструменты», логотип с градиентом, частицы с соединениями.
   - About: нейтральное описание организации без упоминания «один человек» и «разработчики».
   - GitHub: кнопка-ссылка на `https://github.com/starframe-dev`.
   - Footer: копирайт `© 2026 Starframe`.
5. Стили: чистый CSS, глубокий тёмный фон с мягкими градиентами, плавные fade-in анимации, hover-эффекты, адаптив.
6. Язык контента: русский.
7. Без проектов, без блога, без сложной навигации.
8. Проверить `pnpm dev`, `pnpm build`. В результате — статические HTML/CSS в `dist/`.

## Критерии приёмки

- [x] `pnpm dev` запускает сайт без ошибок.
- [x] `pnpm build` завершается успешно и генерирует `dist/index.html`.
- [x] Страница адаптивна: корректно выглядит на десктопе и мобильном.
- [x] На странице есть название `Starframe`, описание деятельности и ссылка на GitHub.
- [x] Проекты не отображаются на сайте.
- [x] Нет упоминания «проект одного человека» и «разработчики».
